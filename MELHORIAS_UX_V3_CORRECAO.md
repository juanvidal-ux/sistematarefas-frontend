# Correção UX V3 - Kanban

Esta versão corrige o problema visual em que o cabeçalho das colunas do Kanban ficava sobreposto aos cards das tarefas.

## Correções aplicadas

- Removido comportamento `sticky` dos títulos das colunas.
- Corrigida sobreposição entre cabeçalho e cards.
- Mantido o visual moderno da V2.
- Removido o selo temporário "Design moderno ativo" da topbar.
- Ajustada estabilidade visual do Kanban em desktop e mobile.

## Arquivo adicionado

- `css/ux-fix-kanban-v3.css`

## Segurança

Não foram alteradas rotas, funções JavaScript principais, IDs, login, permissões ou integração com API.
