# Correção UX V4 - Modal de Subitens

Esta versão corrige o problema em que os subitens ficavam difíceis de visualizar dentro do modal da tarefa.

## Alterações feitas

- Modal ampliado para melhor aproveitamento da tela.
- Área central do modal com rolagem controlada.
- Rodapé de ações deixou de cobrir o conteúdo.
- Formulário de subitens ficou mais compacto.
- Dicas dos campos foram ocultadas dentro do formulário para ganhar espaço.
- Lista de subitens ficou mais próxima do formulário.
- Ajustes de responsividade para telas médias e pequenas.

## Segurança

Não foram alterados:

- IDs do HTML.
- Funções JavaScript.
- Rotas da API.
- Login.
- Permissões.
- Regras do Kanban.
- Estrutura dos dados.
